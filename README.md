# myMTH418
Coursework and implementations for MTH418: Inference I (Estimation Theory &amp; Mathematical Inference).

## MTH418 – Inference I

This repository contains coursework, experiments, and implementations related to **MTH418: Inference I**.  
The course focuses on **Estimation Theory and Mathematical Inference**, which are core areas of Mathematical Statistics.

The goal of this repository is to:
- Practice theoretical concepts through computational implementation.
- Develop small tools and functions (mainly in **R**) for statistical inference.
- Build an organized reference for important ideas in estimation and hypothesis testing.

---

## Topics Covered

This repository may include implementations and notes related to:

- Exponential Families  
- Sufficiency and Minimal Sufficiency  
- Completeness  
- Unbiased Estimation  
- UMVUE (Uniformly Minimum Variance Unbiased Estimator)  
- Information Inequalities (e.g., Cramér–Rao Lower Bound)  
- Methods of Estimation  
- Hypothesis Testing  
- Neyman–Pearson Lemma  
- UMP and UMPU Tests  
- Likelihood Ratio Tests  
- Confidence Intervals

## Purpose

This project is mainly for **learning and experimentation**.
